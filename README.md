# RS-Airy-BM4-mount
This is a repo for the 3d printed mount we made to attach a RS Airy Lidar to the BM4 mounting system. 

You will need the following hardware:
3 M3x6mm countersunk screws
1 #6-32x1 1/4" countersunk screw
2 #6-32x1 1/2" machine screw
3 #6-32 nuts

Included in this repository are the .3mf files for both parts of the mount. I heavily reccomend printing the bm4 plate with the screw face on the print bed so that the layer lines are not perpendicular to the screw holes.
